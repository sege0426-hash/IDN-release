# Manual Screenshot Structure

Place screenshots under role-specific folders and then by major feature category.

## Roles

- `super-admin`
- `group-admin`
- `user`

## Categories

- `super-admin`: `login`, `dashboard`, `management`, `security-policy`, `system-quality`, `admin-settings`, `audit`
- `group-admin`: `login`, `dashboard`, `management`, `admin-settings`, `audit`
- `user`: `login`, `home`, `network`, `member`, `request`, `settings`

## File Naming

Use the manual section or article id as the image file name.

Examples:

- `super-admin/login/2-1-로그인.png`
- `super-admin/management/5-2-3-1-vnet-목록.png`
- `group-admin/admin-settings/5-3-관리자-설정.png`
- `user/network/5-2-네트워크.png`

1. 개요

본 문서는 그룹 관리자 권한으로 접근 가능한 메뉴만 기준으로 정리한 매뉴얼 초안이다.

그룹 관리자는 자신의 소속 그룹 범위 내에서 관리 기능을 수행하며, 최고 관리자 전용 메뉴인 조직 관리, 라이선스 관리, 보안 및 정책, 시스템품질은 포함하지 않는다.

2. 로그인

그룹 관리자는 로그인 화면에서 계정 ID와 비밀번호를 입력해 접속한다. 로그인 이후 2단계 인증 또는 비밀번호 재설정이 필요한 경우 해당 화면으로 이동한다.

2.1 로그인

▶메뉴 위치: 로그인 화면

▶기능 설명:

그룹 관리자 계정으로 시스템에 접속한다.

▶사용 절차:

1. 로그인 화면에 접속한다.

2. 아이디와 비밀번호를 입력한다.

3. 로봇 확인 항목을 선택한다.

4. 로그인 버튼을 클릭한다.

5. 인증 결과에 따라 다음 화면으로 이동한다.

▶입력 항목:

항목 설명 입력 가이드
아이디 그룹 관리자 계정 ID 필수 입력
비밀번호 계정 비밀번호 필수 입력

▶처리 결과 / 유의사항:

로그인 실패 시 오류 메시지가 표시된다.

최초 로그인 시 비밀번호 재설정 화면으로 이동할 수 있다.
